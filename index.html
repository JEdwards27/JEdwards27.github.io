Latest GitHub projects

Distributed Login (Version 1.0)

  • My current final year project which is now a full release. It's a two-factor authentication system that distributes the key accross MySQL databases and a USB device.

  • Therefore if one database is compromised, the entire authentication key is not compromised, as part of it is held statically on a USB device

  • It is also designed to combat Man-in-the-Browser (MitB) attacks by changing the key at sign out.

SingleSiteApp

  • A small python script which will display a pre-determined URL as a full screen web page.

  • Designed to be a direct interface into SoC web interfaces, with a minimalist appearnace, by not having a URL bar or navigation buttons.

  • Handles HTML, which is ideal for use with solutions such as the ELK stack, which has a web interface for monitoring data. (CSS not supported)
