About the project

overview

Project aims
  • Automated backup system with security through port knocking
  • Creating more complex security with no extra input from the user
  • Perform similarly or better than current data backup solutions
At a glance
  • Directory that needs backing up is zipped up and hashed
  • Hash is stripped of all alpha characters, leaving just numeric
  • First 6 numbers are port|count paramters of knocking sequence
  • Zip uploaded to backup server, same operation is performed there
  • Backup server now knows the expected sequence for the next connection
    • GitHub page
Further development
  • > Need to perfect sequence generation through hashes, currently metadata is influencing hash output
  • > Implement TLS for file transfer so that data is encrypted in transit
  • > Implement a test of port knocking in Python to assess feasbility of its automation
