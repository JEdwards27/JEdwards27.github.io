About the project

overview

Project aims
  • Become a go to source of vulnerability related news on Twitter
  • Tweet out useful information from CVE sites in an easily readable form
  • Have multiple different tweet formats for various reader feedback driven features
At a glance
  • First form:
  • Scrape JSON and HTML from cvedetails.com
  • Tweet top 3 CVEs, with 8+ CVSS, exploit count, and url
  • Second form (WIP):
  • Scrape data from exploit databases for latest types of attack
  • Tweet top 3 of attack types that week, e.g. 1:DDOS, 2:RAT etc
  • Note: Follow me on twitter @jacksec27 for updates from this bot
    • GitHub page
Further development
  • > Start second form: Research sources for attack types and how to format
  • > Continuous integration of reader requested features into First Form.
  • > Perhaps a feature that will respond to tweets mentioning the bots account
