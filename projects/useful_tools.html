About the project

overview

Project purpose
  • Tools that I have developed for various different purposes
  • There is no structure to this repository, it's a collection of ideas
  • Beware: A lot of these ideas aren't fully developed
  • Some projects have left this repository and become their own project, see my SMS/Email 2FA project as an example
At a glance
  • RSA Calculator: A script that helps calculate the various elements of RSA, designed mainly for speeding up challenges that requies this in CTFs.
  • Single Site App: A python based web client that display a pre-determined addresses web page, useful for SOC monitroing software that displays data through web interfaces.
  • Status Script: A simple bash script that can tell you useful status information about your debian based Linux box.
  • Note: Click the vertical headings to go to that project

RSA Calculator

Features
  • Works on Python2.7
  • Allows you to enter each part of RSA calculation: n, m, c and tot
  • D part of calculation is still a work in progress

Single Site App

Features
  • Dedicated web interface display app, no search abr or buttons
  • Ideal for keeping tabs on a SoC dashboard for example
  • Does not support CSS or HTML5 or any dynamic web plugins
  • Disclaimer: This project contains borrowed code from this tutorial

Status Script

Features
  • Bash script for displaying hardware and configuration info for linux
  • Currently only tested on Ubuntu distros(Ubuntu 14, 16 & 18)
Further development
  • > RSA Calculator needs D part of RSA calculation, currently prooving difficult to implement mathematically
  • > Single Site App needs to support more screen resolutions
  • > Status script needs to be adapted to work on other linux flavours
